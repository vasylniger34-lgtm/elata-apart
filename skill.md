## 🔍 Фаза 1: Brand DNA (Глибокий аналіз)

> Якщо є аналіз з Wire Code бота (Gemini) — використай як базу і поглиб.

1. **Айдентика**: Кольори (витягнути з фото), шрифтовий настрій, голос бренду.
2. **Атмосфера**: З фото/відео бренду — освітлення, текстури, емоції → перенести в CSS/дизайн.
3. **Авто-стиль**: Обрати напрямок (Luxury Dark / Soft Premium / Industrial / Modern Sharp тощо) — обґрунтувати чому саме цей.
4. **Палітра**: 3-4 HEX кольори з матеріалів бренду + акцентний колір.
5. **Типографіка**: 2 Google Fonts — один для заголовків, один для тексту.
6. **Конкуренти**: Знайти 2-3 конкуренти → визначити їх слабкі місця → зробити краще.
7. **3 унікальні фішки**: Що конкретно робимо в демо (анімація, конфігуратор, квіз, 3D тощо).

## 💎 Стандарти Фронтенду (High-End UX)

- **🎨 ui-ux-pro-max ОБОВ'ЯЗКОВО**: Перший крок — генерація дизайн-системи через скілл `ui-ux-pro-max` (стиль, кольори, шрифти, UX-правила).
- **🧩 21st.dev ОБОВ'ЯЗКОВО**: Для кожної секції (Hero, Navbar, Features, тощо) — шукати готові преміум-компоненти на https://21st.dev/home перед тим як писати з нуля.
- **Унікальність**: Кожен блок адаптований під конкретний бренд на основі дизайн-системи.
- **Анімації**: Мікро-взаємодії, плавні появи (GSAP або чистий CSS), паралакс, Glassmorphism.
- **Шрифти**: Преміальні комбінації з Google Fonts (рекомендовані ui-ux-pro-max: Outfit, Playfair, Montserrat, DM Sans тощо).
- **Мобільна версія**: App-like feel — має виглядати як окремий додаток.
- **Прелоадер**: Обов'язково — вступна анімація або логотип-перехід.
- **SEO meta теги**: Завжди підключати title, description, og:image.

## ⚙️ Технічний стек (Default)

- **Базовий**: Vite + HTML / CSS / Vanilla JS
- **Якщо потрібен складний стейт**: React (Vite)
- **Анімації**: GSAP (основний), CSS Animations (легкі ефекти)
- **Стилізація**: CSS Custom Properties (design tokens), БЕЗ Tailwind/Bootstrap якщо не вказано явно
- **Шрифти**: Google Fonts (завжди підключати)
- **SEO**: Semantic HTML5 + meta теги

## 🗂 Робота з Media Archive (wire_code_media.zip)

Якщо в директорії проекту є ZIP-архів з медіа (скачаний Wire Code Media Grabber ботом):

1. **Сканування**: Знайти всі файли `.jpg`, `.jpeg`, `.png`, `.webp`, `.mp4` в архіві.
2. **Відбір для сайту**: Обрати фото, що найкраще передають атмосферу бренду:
   - Пріоритет: фото продукту/послуги в дії, інтер'єр/екстер'єр, команда
   - Виключити: скріншоти, банери з текстом, низька якість, дублікати
3. **Використання**: Скопіювати відібрані фото в `public/images/` або `src/assets/` проекту.
4. **Оптимізація**: Вказати в коді `loading="lazy"` для всіх зображень нижче першого екрану.
5. **Hero секція**: Найкраще фото — в Hero. Якщо є відео — використати як фонове відео (muted, autoplay, loop).
